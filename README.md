# Books Frontend

Angular frontend for the Books API application.

## Tech Stack

- Angular 20.3 
- Bootstrap 5.3
- FontAwesome 7.2
- RxJS
- JWT Authentication

## Prerquisites

- Node.js 20+

## Installation

`git clone https://github.com/JackHoffsten/books-frontend.git`

`cd books-frontend`

`npm install`

## Development

`ng serve`

## Features

- User registration and login
- JWT authentication with refresh tokens
- Book management (CRUD)
- Quote management (CRUD)
- Dark mode support
- Responsive design
